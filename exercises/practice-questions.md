# PHP OOP Practice Questions

## Section A: Short Answer Questions

### Q1. What is Object-Oriented Programming (OOP)?
**Answer:**  
OOP is a programming paradigm based on objects that contain data (properties) and methods (functions).

---

### Q2. What is a class?
**Answer:**  
A class is a blueprint or template used to create objects.

---

### Q3. What is an object?
**Answer:**  
An object is an instance of a class.

---

### Q4. Name the four pillars of OOP.
**Answer:**  
1. Encapsulation  
2. Inheritance  
3. Abstraction  
4. Polymorphism  

---

### Q5. What is encapsulation?
**Answer:**  
Encapsulation hides internal data and allows access only through public methods.
